# TTool_CTW
TTool_CTW - це програма для локалізації гри Craft The World,<br/>
головною метою створення цієї програми був переклад гри українською мовою

Програма дозволяє редагувати файли локалізації гри (*.csv файли), а також вміє розпаковувати та запаковувати назад ігрові шрифти
